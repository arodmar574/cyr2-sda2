BLOQUE MIENTRAS

Este bloque es un bucle que ejecuta su código mientras se cumpla una condición (True). El bloque se ve así:

El bucle mientras tiene una condición que se debe cumplir para que se ejecute el código. La condición se comprueba antes de empezar el bucle. Si la condición es falsa desde el principio, el código del bucle no se hace nunca.

El bloque solo nos deja poner la condición como True o False, lo que es muy limitado. Podemos hacer cosas más interesantes si usamos condiciones más complejas con los operadores. Estos bloques están en el menú ‘Lógica’ y funcionan igual que en Python -> Introducción. No vamos a ver esto ahora, pero lo mencionamos para que lo sepas.

Vamos a ver un ejemplo de cómo hacer una línea diagonal en los LEDs [(0,0) - (1,1) - (2,2) - (3,3) - (4,4) y (5,5)] de la pantalla. En la imagen se ve el programa completo, donde se ve que la condición es verdadera hasta que index llega a 4, así que se va a ir iluminando cada LED cada segundo. Esto solo pasa cuando se enciende el dispositivo, pero eso no importa para este ejemplo.

BLOQUE CADA

Este bloque ejecuta el código en segundo plano cada cierto tiempo que se puede indicar. El bucle espera ese tiempo antes de volver a ejecutar el código. Es parecido a un bucle “forever” (por siempre), pero con una pausa entre cada ejecución. El bloque es muy útil si, por ejemplo, queremos comprobar cada poco tiempo si pasa algo, como que se pulse una tecla, se oiga un sonido, etc.

BLOQUES SALIR Y CONTINUAR

Con este bloque se puede hacer que el código se repita en segundo plano cada vez que pasa un tiempo que se puede elegir. El bucle hace una pausa de ese tiempo antes de hacer el código otra vez. Es similar a un bucle “forever” (por siempre), pero con un intervalo de tiempo entre cada repetición. El bloque es muy útil si, por ejemplo, queremos ver cada cierto tiempo si ocurre algo, como que se presione una tecla, se escuche un sonido, etc.
