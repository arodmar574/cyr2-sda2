Creación de variables

Creación de variables

Las variables son importantes para usar bucles, y por eso vamos a ver cómo se manejan en MakeCode. Las variables son espacios en la memoria donde el programa guarda valores. Podemos ponerles un nombre como “temperatura”, “velocidad”, “estado”, etc. para usarlos más fácilmente.

El menú ‘Variables’ tiene este aspecto cuando lo abrimos por primera vez y no hay ningún bloque en el área de programa:

Vamos a ver cómo se hace para crear una variable, cambiarle el nombre, borrarla y otras cosas que podemos hacer con ellas. En la siguiente animación se muestra el proceso de crear y nombrar una variable. Al pulsar el botón se abre una ventana donde nos pide el nombre, que escribimos siguiendo las reglas establecidas, y al darle a ‘Aceptar’ nos aparecen tres bloques, dos para usar la variable creada y uno que es la variable misma, todos con el nombre que le hemos puesto a esa variable.

El bloque ‘fijar’ asigna un valor inicial a la variable, el bloque ‘cambiar’ suma el valor que pongamos en ‘por’ a la variable, como si usáramos el operador ‘+=’ y el último bloque es el valor actual de la variable.

También podemos ver que al pulsar la flechita de cualquier bloque se abre un menú donde aparece la variable, y donde podemos crear otra nueva, cambiarle el nombre o eliminar la variable que está seleccionada en ese momento. En la siguiente animación se muestra cómo hacer estas cosas.

Los bloques ‘fijar’ y ‘cambiar’ no se crean de nuevo cuando hacemos nuevas variables, sino que podemos elegir la variable que queramos en el desplegable. Pero sí se crea un bloque nuevo para el valor de cada variable. También podemos ver que si borramos una variable que está usada en varios sitios, se borra, con un aviso previo, de todos ellos.

Pero las variables no se borran del menú ‘Variables’, donde siguen disponibles hasta que las borremos desde algún bloque en la zona de programa.

Reto 9: Brújula

Se trata de un programa que en función de la posición de la brújula de microbit nos indique en los leds en modo texto la dirección hacia la que apunta. Norte (N), Este (E), Sur (S), Oeste (O). Hay que tener en cuenta los ángulos en microbit.

Contenidos trabajados:

  • Entorno de programación Makecode
  • Envío de contenidos a microbit
  • Bucles (repetir, por siempre)
  • Condicionales
  • Creación y uso de variable
  • Entradas con brújula

Condiciones del ejercicio:

  • Emplea el bloque establecer ángulo para obtener la dirección de la brújula
  • Utiliza condicionales para determinar las diferentes coordenadas teniendo en cuenta entre qué ángulos están.

Reto 10: Contador de turno

En esta actividad tendrás que realizar el control de turno de una carnicería o punto de venta, si pulsamos el botón “A” suma 1, y si pulsamos el botón “B” resta 1. El programa cuando empiece tendrá el contador a 0, y el número del turno no puede ser negativo.

Contenidos trabajados:

  • Entorno de programación Makecode
  • Envío de contenidos a microbit
  • Bucles
  • Creación y uso de variable
  • Trabajo inicial con eventos
  • Entradas con pulsadores

Condiciones del ejercicio:

  • Debes hacer que el programa establezca el contador a 0 antes de empezar a contar.
  • Al presionar un botón debe sumar 1.
  • Al pulsar el otro botón debe restar 1.

  • Entorno de programación Makecode
  • Envío de contenidos a microbit
  • Bucles (Para y Mientras)
  • Creación y uso de variable
  • Tendrás que crear las variables "X" y "tiempo"

  • Deberás establecer un tiempo.
  • Tendrás que darle a X un valor de 0 a 4.
  • Graficar X e invertir X estableciendo pausas entre ellos.
